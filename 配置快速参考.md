# 飞书云表格配置快速参考

## 🚀 快速开始（3步完成）

### 步骤1：创建飞书应用并获取凭证

1. 访问 https://open.feishu.cn
2. 创建企业自建应用
3. 获取 **App ID** 和 **App Secret**
4. 添加权限：`bitable:app`、`drive:drive`

### 步骤2：创建多维表格并获取信息

1. 在飞书中创建多维表格
2. 添加字段：
   - 教案名称（文本）
   - 教案编号（文本）
   - 教案评价（多行文本）
   - 修改意见（多行文本）
   - 错别字（文本）
   - 状态（单选：待审核、已审核、已修改、已完成）
3. 从浏览器URL中提取：
   - **App Token**：`base/` 后面，`?` 前面的部分
   - **Table ID**：`table=` 后面的部分

### 步骤3：配置环境变量

**使用配置助手（推荐）：**
```bash
cd backend
node setup-lark.js
```

**或手动配置：**
```bash
cd backend
cp .env.example .env
# 编辑 .env 文件，填入配置信息
```

## ⚠️ 常见问题

**Q: 如何获取 App Token 和 Table ID？**

A: 打开多维表格，查看浏览器地址栏：
```
https://xxx.feishu.cn/base/AppToken123?table=TableID456
                                    ↑              ↑
                                 App Token    Table ID
```

**Q: 配置后仍然显示"模拟模式"？**

A: 
1. 检查 `.env` 文件是否在 `backend` 目录下
2. 确认所有配置项都已填写（没有留空）
3. 重启后端服务
4. 检查权限是否已通过审批

**Q: 创建记录失败？**

A:
1. 检查字段名称是否与表格中的完全一致
2. 确认应用权限已添加并通过审批
3. 检查表格权限设置
